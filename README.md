### 🌟 프로젝트 주제

- ‘당근이지’는 취미를 공유하고 싶은 이웃을 찾는 사용자를 위한 웹 페이지 입니다.
- 이 제품은 사용자가 게시글을 작성하고 검색할 수 있지만, 일반 게시판과는 달리 카테고리 별 게시글 작성을 통한 관심사 공유가 차별화의 포인트 입니다.
